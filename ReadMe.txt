۱: فاصله های اضافی و خط های خالی را حذف میکنیم
۲: آدرس های اینترنتی و لینک ها را حذف میکنیم
۳: تگ های HTML را حذف میکنیم
۴: اعداد و حروف انگلیسی (لاتین) را حذف میکنیم
۵: اعلام نگارشی را حذف میکنیم
۶: هشتگ ها را حذف میکنیم
۷: بدلیل حذف برخی کارکتر ها فضا های خالی جدید ایجاد شده اند و آنها را حذف میکنیم
۸: تمام متن را توکن بندی میکنیم
